# Heijunka Toyota Packing Solver

Web-based packing distribution solver dengan tema Toyota Manufacturing.
Digunakan untuk menyeimbangkan target packing harian berdasarkan pola produksi.

## Fitur
- Input bulk part & packing pattern
- Auto leveling box & pieces
- Visual dashboard (Before vs After)
- Export Excel (Multi-sheet)
- MSP Format Output

## Cara Menjalankan
```bash
pip install -r requirements.txt
streamlit run app.py
